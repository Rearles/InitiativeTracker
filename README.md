# Initiative Tracker

Initiative Tracker for my own personal use with my DnD campaigns. 

## Installation

Use the package manager [Node]https://nodejs.org/en/) to install the neccessary thigns for Initiative Tracker.

```bash
npm install 
```

## Usage

To run the project to localhost type the following command

```bash
ng serve
```
